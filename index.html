spGoogle

Development package for R users. The spGoogle package provides a interface between R and GoogleMaps for data visualization in both directions. Import GoogleMaps background to R plots and export R data information to GoogleMaps. Currently, on Windows the spGoogle can only export to Google Chrome browser while on Linux both Google Chrome and Firefox are supported.

Installation

You can install spGoogle from github using the devtools package

install.package("devtools")
devtools::install_github("lgsilvaesilva/spGoogle")

Example

require(spGoogle)
data(rio)

spRplot(rio) #plot in R
spGplot(rio) #plot in Google Maps ou Google Earth

spGplot(rio, var = "Dengue")
